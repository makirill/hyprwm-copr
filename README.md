# HyprWM Copr Packaging

RPM spec files and related packaging resources for building [HyprWM](https://hyprland.org/) and its ecosystem packages for Fedora Linux.

**Copr Repository:** https://copr.fedorainfracloud.org/coprs/makirill/hyrpwm/

This repository contains the source files used to build and maintain HyprWM packages in the Copr build system, making it easier for Fedora users to install and use the Hyprland Wayland compositor.
